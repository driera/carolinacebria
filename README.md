# Carolina Cebriá — Photography Portfolio

Personal photography portfolio website for Carolina Cebriá, showcasing her work across several projects.

## Infrastructure

| Service | Provider |
|---|---|
| Domain registrar | Raiola Networks |
| DNS | Cloudflare |
| Hosting | GitHub Pages |
| Email | Current hosting (migration to Zoho Mail pending) |

## Deploy

Pushing to `main` triggers an automatic deploy via GitHub Actions (`.github/workflows/deploy.yml`).

## CSS

`css/main.css` is the original minified stylesheet. New styles go in `css/styles.css`, which will gradually replace it.
